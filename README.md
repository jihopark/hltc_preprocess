# hltc_preprocess
Preprocessing library for our lab's use

## Setup
`pip install hltc_preprocess`

## Modules
1. vocabulary.py
	- `create_vocabulary`
	- `idx_tokens`
	- `find_phrases`
2. `tweets.py`
	- `clean_tweet`
	- `filter_tweet`
	- `tokenize_tweets`
3. `seq2seq.py`
	- `add_padding`
	- `add_position_token`
